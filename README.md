## Features
### WebP Next Generation Optimized Images

Not long ago most websites actively used the outdated PNG and JPG format images which are big in size and correspondingly take some time to load. Unfortunately, now there is not much time to spare for page loading since it can lead to customers leaving. 

Fortunately, Google came up with the next generation file format - WEBP that is considerably smaller than PNG and JPG that loads faster with no delay. Magento 2 Image Optimization Extension will help you to convert all current and future images into the WEBP format.

### Up to 50% File Size Reducing

The best thing about Magento 2 Image Optimization extension is that it allows you to choose the quality and size of images you want them to be converted to. You can reduce images size up to 50% not influencing their quality significantly.

### Improved Website Performance

The smaller the size of the images the smaller the size of the page in general and the faster it will be loaded. Magento 2 Image Optimization extension will not only help you to convert image into the modern file format, but lazy load them generally reducing the page size and improving your website performance.

### Better SEO Ranking

If you’re looking for the extension to optimize your Magento 2 website images then you probably know how website performance influences SEO. If the website takes longer than 3 seconds to load your visitors will most probably leave and that can reflect dramatically on your bounce rates. Image optimization extension will help you optimize images and correspondingly improve you SEO.

<p align="center">
  <img width="700" height="400" src="https://cm.magefan.com/mf_webp/png/media/archive/improved-seo-ranking.webp">

### Multi-Browser Compatibility

WEBP file format was developed and is used by Google. So, it is generally accepted by a lot of browsers, but not all of them. There are some that haven’t yet integrated the WEBP format, However, you don’t need to worry about what images people from those browsers will see entering your website. 

Magento 2 WebP Images extension as one of the bundle products of Magento 2 Image Optimization module creates the copy of your PNG and JPG images. Thus, when people come to your website from some browsers that aren't compatible with WEBP they will see your original images.

</p>

### Complex Magento Image Optimization Solution

Magento 2 Image Optimization Extension is a bundle of 2 products: WebP Images and Image Lazy Load. Therefore it is a complex image optimization solution that will help you to optimize your website performance, reduce loading time and improve SEO. 

<p align="center">
  <img width="700" height="349" src="https://cm.magefan.com/mf_webp/png/media/archive/complex-magento-image-optimization-solution.webp">

### Automated Image Optimization Processes

It wouldn't be very convenient to provide you with the solution you’d need to update from time to time. Magento 2 Image Optimization works automatically. Once you set the configuration images will be automatically converted to the WEBP format and image lazy load will be triggered automatically once someone scrolls down the page. That way you will have your website performance improved automatically with Magento 2 image optimization extension.
